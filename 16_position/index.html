[position 익히기]

position은 모든 태그 요소들을 공중에 띄어서 원하는 위치에 배치하는 css 명령
가장 많이 사용되는 css명령은 3가지입니다

  1. static:기본값으로 position명령을 해제
  2. position:relative- 1. 제자리에서 공중으로 띄우기 2. 상대적인 기준을 설정
  3. position:absoulute -> 요소를 원하는 위치에 배치기준은 웹브라우저
  4. left, right, top, botton으로 위치를 배치한다 좌우1, 상하1
  5. z-index -> 공중에 띄어져 있는 position끼리 우선순위
